## Attendance-check-Guide

### System
부산소프트웨어마이스터고등학교 학생증의 NFC를 이용해 학생증을 찍으면 고유 코드가 DB로 이동하여 웹 페이지에 보여주는 출석관리 시스템 <br>

### Commit Rules

| type | description |
| --- | --- |
| ADD ::  | 기능 추가 |
| FIX :: | 수정 |
| DOCS :: | 세부 기능 추가 |
| STYLE :: | 코드 스타일 변경 |
| REFACTOR :: | 코드 리팩토링 |
| TEST :: | 코드 테스트 |
| DELETE :: | 파일 또는 기능 삭제 |
